# This Repository contains Python scripts (Jupyter Notebooks) for A/B Testing methods for Binary Outcome, Continous Outcome Metrics.

The first notebook has Testing methods for Binary Outcome metric (ChiSquare Test, Fisher's Exact Test).
The second notebook has Testing methods for Continous Outcome metric (T-Test, Welch Test, Mann Whitney U Test).
And the third notebook has OneWay and TwoWay Annova tests illustrated.

### Dataset Sources:
dataset(ab_data.csv) url: https://www.kaggle.com/datasets/zhangluyuan/ab-testing
dataset(cookie_cats.csv) url: https://www.kaggle.com/datasets/yufengsui/mobile-games-ab-testing
dataset(countries.csv) url: https://www.kaggle.com/datasets/putdejudomthai/ecommerce-ab-testing-2022-dataset1
